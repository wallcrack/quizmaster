导入题库

支持 PNG / JPG / JPEG / GIF / WebP，可一次选择多张图片。 导入文件中用 image: 文件名.png 引用，题干中用 ![描述](文件名.png) 显示。

文件格式示例（YAML）：

questions:
  - type: single
    content: "Python 中列表推导式的关键字是？"
    options: ["for", "if", "in", "all of the above"]
    answer: "0"
    explanation: "列表推导式使用 for 关键字。"
    difficulty: easy
    source: "Python Cookbook"
    chapter: "第 2 章"
    tags: ["python", "语法"]

带图片的题目示例：

questions:
  - type: single
    content: "![架构图](cpu_architecture.png)\n如图所示，冯·诺依曼架构的核心部件是？"
    options: ["运算器、控制器、存储器", "运算器、输入设备、输出设备"]
    answer: "0"
    image: cpu_architecture.png
    tags: ["计算机基础"]

Markdown 格式示例：

---
type: single
difficulty: easy
source: Python Cookbook
chapter: 第 2 章
tags: [python, 语法]
options:
  - for
  - if
  - in
  - all of the above
answer: "0"
explanation: 列表推导式使用 for 关键字。
---
题干内容（支持 Markdown 和 LaTeX 公式）
---
type: subjective
difficulty: medium
tags: [网络]
---
简述 HTTP 与 HTTPS 的区别。
---

说明：

  • 题干和解析支持 Markdown 格式
  • LaTeX 公式用 $ 或 $$ 包裹，例如 $E=mc^2$
  • 导入文件中的 image 字段引用图片文件名
  • 题干中用 ![描述](文件名.png) 语法内联显示图片
  • 图片先保存到服务器，再导入题目，确保引用有效
